#+title: REPL

All of the relevant help for the REPL can be accessed from inside the REPL, for
completeness though, the help strings are reproduced here with the =data= command
macro.

#+begin_src @example
data`help`
#+end_src

#+begin_src @example
data`?add`
#+end_src

#+begin_src @example
data`?delete`
#+end_src
